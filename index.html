🧠 TECFE v6 IA

Teste Eletrônico de Classificação de Cartas para Funções Executivas

Versão computadorizada do TECFE (Teste Eletrônico de Cclassificação de Cartas para Funções Executivas), Baseado no Wisconsin Card Sorting Test (WCST) e, como o nome informa, ele serve avaliação das funções executivas: flexibilidade cognitiva, controle inibitório, formação de conceitos e memória de trabalho.

  • 16 indicadores calculados automaticamente (Heaton et al., 1993)
  • Normas brasileiras (Silva-Filho, 2007) e americanas (Heaton, 2005)
  • Parecer clínico assistido por Inteligência Artificial (Gemini)
  • Detecção automática de resposta aleatória
  • Relatório PDF completo com gráficos e análise clínica
  • Histórico cumulativo com comparativo entre execuções
  • Recuperação automática em caso de queda de energia
  • Atualização automática via internet
  • 3 execuções demo gratuitas para avaliação
  • Aplicativo standalone — não requer navegador

Requisitos: Windows 10/11 (64-bit)
